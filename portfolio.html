1.1.9 BBQ Burger

BBQ Burger is a burger customization game where the user can create a burger with a variety of vegetables. This game was created using user input and if/elif statements. If a user enters an answer that is unsupported, the program becomes irresponsive and allows the user to start again and create their burger.


1.2.5 Berry Collector

Berry Collector is a Pokemon-related game where Pikachu is trying to obtain as many berries as possible within a one-minute time limit while trying to avoid the moving Pokeballs. This game was created using functions, if/elif statements, and allowed for user input. If a user runs into a moving Pokeball, the program immediately terminates.


Trench Run

Trench Run is a remixed an obstacle plane flying game. We took inspiration for the trench run in Star Wars and remixed the game to fit the theme. In this game the player is trying to get through the three easy levels and progress to hard mode by flying the X-Wing into the holes in the walls. This game uses procedures, animation, counters, costume changes, events, input, and sound.


2.1.6 A phishy Fish Tank

Within a phishy Fish Tank, we had to debug a piece of code which prevents the file from running. There were many errors regarding alkalinity, phoshpates, temperature, calcium and magnesium. Some errors were based around values being too high/low, for loops were incorrect and more. When everything is corrected, the user can check the factors of the fish tank to make sure it is suitable for the fish.

4.1.4 Understanding Complex Systems

With Understanding Complex Systems, two different simulations were used to simulate real world issues regarding different situations such as climate change, war, bidding market, traffic intersection, taxi cabs, minority game, segregation, prisoners dilemma, turbulence, and more.Looking through a Climate Change and a Rebellion simulator allows the user to understand the model, connect it to world issues, understand bias, and figure how different facotrs can affect the things at hand.
