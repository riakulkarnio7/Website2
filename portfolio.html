1.1.9 BBQ Burger

BBQ Burger is a burger customization game where the user can create a burger with a variety of vegetables. This game was created using user input and if/elif statements. If a user enters an answer that is unsupported, the program becomes irresponsive and allows the user to start again and create their burger.


1.2.5 Berry Collector

Berry Collector is a Pokemon-related game where one main turtle(Pikachu) is trying to obtain as many berries as possible within a one-minute time limit while trying to avoid the moving Pokeballs. This game was created using functions, if/elif statements, and allowed for user input. If a user runs into a moving Pokeball, the program immediately terminates.
