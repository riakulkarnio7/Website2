1.1.9 BBQ Burger

BBQ Burger is a burger customization game where the user can create a burger with a variety of vegetables. This game was created using user input and if/elif statements. If a user enters an answer that is unsupported, the program becomes irresponsive and allows the user to start again and create their burger.
